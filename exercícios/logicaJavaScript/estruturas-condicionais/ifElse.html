Condição if...else

A instrução if...else executa uma instrução se uma condição especificada for verdadeira. Se a condição for falsa, outra instrução na cláusula else opcional será executada.

Sintaxe

        
            if(condição) {
                true
            } else {
                false
            }

Declaração que é executada se a condição for true: Pode ser qualquer instrução, incluindo outras instruções if() aninhadas. Para executar várias instruções, usamos uma instrução de bloco que será declarada com o caractére de chaves, { /* ... */ }, para agrupar essas instruções. Para não executar instruções, use uma instrução vazia.

Declaração que é executada se a condição for false e a cláusula else existir: Pode ser qualquer instrução, incluindo instruções de bloco e outras if() instruções aninhadas.

Várias instruções if...else podem ser aninhadas para criar uma cláusula if() else. Observe que não há palavra-chave if else (em uma palavra) em JavaScript.
