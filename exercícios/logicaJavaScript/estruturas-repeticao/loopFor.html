Loop For

A instrução for cria um loop que consiste em três expressões opcionais, entre parênteses e separadas por ponto e vírgula, seguidas por uma instrução (geralmente uma instrução de bloco ) a ser executada no loop.

Sintaxe

        
            for (variável de inicialização; condição; passo) {
                //executa iteração para o bloco equanto for "true"
                //para a iteração quando for "false"
            }

Loop for in

A instrução for...in itera sobre todas as propriedades de string enumeráveis de um objeto (ignorando propriedades codificadas por símbolos ), incluindo propriedades enumeráveis herdadas, ou sobre elementos de uma array.

Sintaxe

            
                const object = { a: 1, b: 2, c: 3 };

                for (const property in object) {
                console.log(`${property}: ${object[property]}`);
                }

for of

A instrução for...of executa um loop que opera em uma sequência de valores provenientes de um objeto iterável . Objetos iteráveis incluem instâncias de built-ins como Array, String, TypedArray, Map, Set, NodeList (e outras coleções DOM), bem como o arguments objeto, geradores produzidos por funções geradoras e iteráveis definidos pelo usuário.

Um for...ofloop opera nos valores originados de um iterável um por um em ordem sequencial. Cada operação do loop em um valor é chamada de iteração , e diz-se que o loop itera sobre o iterável . Cada iteração executa instruções que podem se referir ao valor da sequência atual.

Quando um loop for...of itera sobre um iterável, ele primeiro chama o método do iterável, que retorna um iterador e, em seguida, chama repetidamente o método do iterador resultante para produzir a sequência de valores a serem atribuídos a variable.

Sintaxe

            
                /* for  ( variável of iterável ) {
                      declaração
                   } 
                */
               

                const array1 = ['a', 'b', 'c'];

                for (const element of array1) {
                console.log(element);
                }
