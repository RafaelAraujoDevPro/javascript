Loop de repetição while

A instrução while cria um loop que executa uma instrução especificada desde que a condição de teste seja avaliada como verdadeira. A condição é avaliada antes de executar a instrução.

Sintaxe

        
            while(condição){
                declaração
                ++ ou --
            }

condição

Uma expressão avaliada antes de cada passagem pelo loop. Se esta condição for avaliada como true , statement é executado. Quando a condição é avaliada como false , a execução continua com a instrução após o while loop.

statement

Uma instrução opcional que é executada desde que a condição seja avaliada como verdadeira. Para executar várias instruções dentro do loop, usamos uma instrução de bloco{ /* ... */ } ( ) para agrupar essas instruções. Observação: usamos a instrução break para interromper um loop antes de condition ser avaliado como verdadeiro.

Essa é a estrutura de repetição que faz o teste lógico no inicio

do...while

A instrução do...while cria um loop que executa uma instrução especificada até que a condição de teste seja avaliada como falsa. A condição é avaliada após a execução da instrução, resultando na execução da instrução especificada pelo menos uma vez.

statement

Uma instrução que é executada pelo menos uma vez e é reexecutada cada vez que a condição é avaliada como verdadeira. Para executar várias instruções dentro do loop, use uma instrução de bloco{ /* ... */ } ( ) para agrupar essas instruções.

Essa é a estrutura de repetição que faz o teste lódico no final
