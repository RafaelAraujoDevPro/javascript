Problema proposto

  • Fazer o Usuário1 digitar um número.
  • Fazer o Usuário2 digitar outro número.
  • Se os dois números digitados forem iguais: mostrar uma mensagem 'Empate!'
  • Se os dois números forem diferentes: realizar um sorteio com os número 0 e 1, e mostrar resultado do sorteio.
  • Se o sorteio der 1, ganha o Usuário que digitou o MAIOR número
  • Se o sorteio der 0, ganha o Usuário que digitou o MENOR número.
  • Então mostrar mensagem informando qual Usuário ganhou
