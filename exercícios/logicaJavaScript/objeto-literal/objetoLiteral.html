Objeto Literal

O que é um objeto literal?

Objeto literal pode ser entendido como uma variável composta que armazena uma coleção de dados. Estes dados são agrupados em pares de um valor sendo referênciado por um nome. Já estes pares podem são chamados de propriedades ou atributos que são associadas ao objeto literal. Estas propriedades podem ser entendidas como variáveis normais que são ligadas ao objeto. Elas podem ser acessada em outras partes do programa externa ao objeto para serem utilizadas.

Um objeto JavaScript é um mapeamento entre keys(chave) e values(valor). Chaves são strings (ou símbolos), e o valor pode ser qualquer coisa. As propriedades podem receber qualquer tipo de dados do javascript como valor; tanto dados primitivos quanto dados por referência. Propriedades não definidas de um objeto tem o valor de undefined (e não null.)

Inicializador de objetos

Um inicializador de objetos é uma expressão que descreve a inicialização de um arquino object. Para iniciar um objeto em um programa de javascript podemos utilizar Object.create(), new Object(), uma função construtora invocada com o operador new ou ainda declarar o objeto com a notação literal.

New Object()

                        
                            let car1 = new Object();
                            car1.color = 'azul';
                            car1.price = 80000;
                            car1.model = 'Gol G7';
                            car1.brand = 'VW';
                            console.log(car1);
                            // run:
                              {
                                color: 'azul',
                                price: 80000,
                                model: 'Gol G7',
                                brand: 'VW'
                              }

Função construtora

                        
                            function Car (color,price,model,brand) {
                                this.color = color;
                                this.price = price;
                                this.model = model;
                                this.brand = brand;
                                this.info = ()=>{
                                    return `Carro ${this.brand} ${this.model} na cor ${this.color} esta custando ${this.price}`
                                };
                            };
                            const car2 = new Car('prata',150000,'Corolla','Toyta');
                            console.log(car2);
                            console.log(car2.info());
                            // run:
                                Car {
                                    color: 'prata',
                                    price: 150000,
                                    model: 'Corolla',
                                    brand: 'Toyta',
                                    info: [Function (anonymous)]
                                }
                                Carro Toyta Corolla na cor prata esta custando 150000

Acessadores de propriedades

Pode-se pensar que objetos se comprotam como uma array matriz associativa. O nome da chave da propriedade funciona como um indice ou indentificador, que é associado ao valor da propriedade. Acessadores de propriedades fornecem acesso às propriedades de um objeto associando elas ao nome da variável usando a notação de ponto '.' ou a notação de colchetes '[]'. Desta maneira podemos acessar as propriedades do objeto fora dele para reatribuição de valor ou então para inseri-las em outro trecho de código do programa.

Exemplo de sintaxe (dot notation)

                            
                                let car = Object.create({});
                                car.color = 'azul';
                                car.model = 'Gol G7';
                                car.brand = 'VM';
                                console.log(car.model);
                                // run:
                                    Gol G7

No exemplo de sintaxe acima foi usada a notação de ponto que é a mais usada para trabalhamos acessando objetos, porém se declararmos car['color'] tambem funcionará normalmente. O nome de uma propriedade de objeto pode ser qualquer string javascript válida ou também qualquer coisa que possa ser convertida para uma string válida, incluindo até mesmo string vazia. No entanto, qualquer nome de propriedade que não seja um indentificador javascript válido (como exemplo, nome de propriedae que contém espaço ou hifén, ou que comece com um número) só poderá ser acessado(a) utilizando a notação de colchetes. Essa notação também é muito útil quando queremos acessar uma propriedade de um objeto de maneira dinâmica (Quando o nome da propriedade não é determinado até o momento da execução).

Exemplo de sintaxe (bracket notation)

                            
                                const user = {
                                    name: 'Rafael',
                                    'last-name': 'Araujo',
                                }
                                console.log(user['last-name']);
                                // run:
                                    Araujo

Já quando precisamos acessar uma propriedade de maneira dinâmica, onde uma string esta armazenada em uma variável, devemos usar a notação de colchetes para receber (dentro), a variável que irá passar o valor de string para nomear a propriedade do objeto.

Exemplo de sintaxe Propriedades dinâmicas

                            
                                const meuCarro = {};
                                const fabricante = 'fabricante';
                                const modelo = 'modelo';
                                meuCarro[fabricante] = 'Ford';
                                meuCarro[modelo] = 'Mustang';
                                console.log(meuCarro);
                                //run:
                                    { fabricante: 'Ford', modelo: 'Mustang' }

Este conceito de propriedade dinâmica é muito útil quando declaramos um objeto literal dentro de uma função pois podemos acessar uma propriedade do objeto utilizando como chave o valor de um parâmetro da função (já que parâmetros de função são variáveis).

Exemplo de sintaxe propriedades dinâmicas em Funções

                            
                                const setAccType = (accType) => {
                                const typeCards = {
                                    platinum: 'Platinum Customer',
                                    gold: 'Gold Customer',
                                    silver: 'Silver Customer',
                                    getAccType(){
                                        return typeCards[accType]
                                    }
                                }
                                return typeCards.getAccType()
                            }
                            console.log(setAccType('silver'));
                            // run:
                                Silver Customer
